# Imagens da Documentação

Esta pasta contém todas as imagens usadas na documentação do Sprite Render Tool.

## Como usar

Para referenciar uma imagem nos arquivos Markdown da documentação, use:

```markdown
![Descrição da imagem](images/nome-do-arquivo.png)
```

Ou com caminho relativo a partir de `en/` ou `pt/`:

```markdown
![Descrição da imagem](../images/nome-do-arquivo.png)
```

## Estrutura sugerida

- `screenshots/` - Capturas de tela da interface do add-on
- `diagrams/` - Diagramas e esquemas explicativos
- `examples/` - Exemplos visuais de uso
- `gifs/` - Animações GIF (quando disponíveis)
