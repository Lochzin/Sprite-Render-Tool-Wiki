# Favicon Personalizado

Para adicionar seu próprio favicon e substituir o ícone "JV" padrão do tema:

1. **Crie ou obtenha um favicon:**
   - Tamanho recomendado: 32x32 ou 16x16 pixels
   - Formatos suportados: `.ico`, `.png`, `.svg`
   - Nome sugerido: `favicon.ico` ou `favicon.png`

2. **Coloque o arquivo nesta pasta:**
   - `docs/assets/images/favicon.ico` (ou `.png`)

3. **O arquivo `_includes/head_custom.html` já está configurado** para carregar o favicon automaticamente.

4. **Se usar um nome diferente de `favicon.ico`:**
   - Edite `docs/_includes/head_custom.html` e altere o caminho do arquivo.

**Dica:** Você pode gerar um favicon online em sites como:
- https://favicon.io/
- https://realfavicongenerator.net/
